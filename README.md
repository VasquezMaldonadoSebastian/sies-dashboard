# 🎓 Mifuturo Datos — Educación Superior Chile

Datasets históricos del **Servicio de Información de Educación Superior (SIES)**,
del Ministerio de Educación de Chile, obtenidos desde [mifuturo.cl](https://www.mifuturo.cl/sies/).

## Contenido

```
mifuturo-datos/
├── matricula/             # Matrícula 2007–2025 (estudiantes matriculados)
├── titulacion/            # Titulación 2007–2025 (titulados)
├── personal-academico/    # Personal Académico 2008–2025
├── oferta-academica/      # Oferta Académica 2010–2026 (carreras y programas)
├── buscadores/            # Datos de buscadores del portal 2025–2026
├── MEMORIA.md             # Registro de sesiones y decisiones del proyecto
└── README.md              # Este archivo
```

## Fuente

Todos los datos son de acceso público, publicados por el SIES
(Subsecretaría de Educación Superior, MINEDUC) en `www.mifuturo.cl`.
